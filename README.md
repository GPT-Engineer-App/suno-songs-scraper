# suno-songs-scraper

A tool that scrapes the info of my own songs on Suno from the Suno api (https://studio-api.suno.ai/api/feed/v2?page=0), and put it in a browser searchable database.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/suno-songs-scraper.git
cd suno-songs-scraper
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
